# 多機能お絵描きアプリ (Simple Drawing App) 🎨

ブラウザだけで動く、高機能でシンプルなお絵描きツールです。
インストール不要で、PCでもスマホでもすぐに使えます。

## ✨ Ver.5 の新機能
以前のバージョンから大幅にパワーアップしました！

*   **↩️ ひとつ戻る (Undo)**: 失敗しても安心。直前の線を10回まで取り消せます（全消去も元に戻せます）。
*   **💾 画像保存**: 描いた傑作を `.png` 画像としてダウンロード・保存できます。
*   **🖼️ 画像読み込み**: スマホの写真やPCの画像を読み込んで、落書きや塗り絵が楽しめます。
*   **🎨 カラーパレット**: よく使う10色をワンタップで切り替えられます。

## 🛠️ 基本機能
*   **✏️ ペン描画**: 滑らかな書き味で自由にお絵描き。
*   **🧹 消しゴム**: 部分的に消せます（ペンの太ことは別に、消しゴムの太さを記憶します）。
*   **📏 太さ変更**: 線の太さを 1px〜50px まで自由に調整可能。
*   **🗑️ 全消去**: キャンバスを真っ白にリセットします。
*   **📱 スマホ完全対応**: タッチ操作に対応。ピンチズームなどは無効化し、描きやすさに特化しています。

## 🚀 使い方
1.  **ブラウザで開く**: `index.html` を開くだけで動きます。
2.  **色を選ぶ**: パレットの丸いボタンか、右側のカラーピッカーで色を決めます。
3.  **描く**: マウスや指で描きます。
4.  **保存する**: 「💾 保存」ボタンで作品をダウンロード！
