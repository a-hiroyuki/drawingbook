# シンプルお絵描きアプリ 🎨

ブラウザで動く、シンプルなお絵描きツールです。
HTML5 CanvasとJavaScriptのみで作られており、特別なインストールは不要です。

## 🌐 デモ

## ✨ 機能
*   **✏️ ペン描画**: 好きな色で自由に絵が描けます。
*   **🧹 消しゴム**: 間違えた箇所を部分的に消せます（太さの記憶機能付き）。
*   **📏 太さ変更**: ペンと消しゴム、それぞれの太さを調整できます。
*   **🗑️ 全消去**: 描いた絵を一度にリセットできます。
*   **📱 スマホ対応**: タッチ操作に対応しており、スマホやタブレットでもスムーズに動きます。

## 🚀 使い方
1.  このリポジトリをダウンロード、またはクローンします。
2.  `index.html` をブラウザ（Chrome, Safari, Edgeなど）で開きます。
3.  すぐに描画を開始できます！
